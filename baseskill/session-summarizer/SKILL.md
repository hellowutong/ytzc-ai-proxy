# Session Summarizer

自动总结对话会话的核心内容和关键结论。

## 使用时机

- 用户结束会话时自动触发生成
- 需要快速回顾长对话的核心要点
- 为会话添加搜索友好的文本描述
- 将对话历史转化为可检索的知识

## 指令

1. **阅读完整对话**
   - 按时间顺序阅读所有消息
   - 识别用户初始问题和最终需求
   - 注意关键转折点和解决方案

2. **提取核心信息**
   - 归纳用户的主要问题或目标
   - 总结 AI 提供的核心答案或方案
   - 列出关键技术点、工具或方法
   - 记录重要的结论或决策

3. **生成结构化摘要**
   - **主题**: 一句话概括会话主题
   - **问题**: 用户提出的核心问题
   - **方案**: AI 提供的主要解决方案
   - **要点**: 3-5个关键知识点
   - **结论**: 最终的结论或行动项

4. **质量检查**
   - 确保摘要不超出长度限制
   - 确认信息准确无遗漏
   - 保持客观中立的表述

## 约束条件

- 摘要长度：100-200字
- 使用中文编写
- 保持客观中立，不添加主观评价
- 突出技术要点和可操作信息

## 输出格式

```markdown
## 会话摘要

**主题**: {一句话主题描述}

**核心问题**: {用户的主要需求或问题}

**解决方案**: {AI提供的主要答案或方法}

**关键要点**:
1. {要点1}
2. {要点2}
3. {要点3}
4. {要点4（可选）}

**技术栈/工具**: {涉及的工具体系}

**结论/下一步**: {最终结论或建议行动}
```
