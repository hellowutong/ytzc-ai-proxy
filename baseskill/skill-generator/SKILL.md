# Skill Generator

根据会话内容生成符合 CC Skill 格式的技能草稿。

## 使用时机

- 用户在会话管理页面点击"生成 Skill"
- 大模型处理完成后自动触发生成
- 从历史会话中提取知识沉淀为可复用技能

## 指令

1. **分析会话内容**
   - 识别核心主题和知识点
   - 提取关键概念和术语
   - 理解用户意图和工作流程

2. **生成 Skill 结构**
   - 编写清晰的 name 和 description
   - 定义精确的指令 (instructions)
   - 规划使用场景 (when to use)

3. **遵循 CC Skill 格式**
   - 符合标准 YAML/JSON 结构
   - 包含必要元数据
   - 便于后续版本管理和发布

4. **输出要求**
   - 生成 draft 状态的 Skill
   - 标记来源会话 ID
   - 保存变更原因

## 约束条件

- 摘要长度不超过200字
- 指令需清晰、可执行
- 使用场景需具体、有边界

## 技能输出示例

```yaml
---
name: {技能名称}
description: {一句话描述}
---

# 技能名称

## 使用时机
当用户需要{具体场景}时使用此技能。

## 指令
1. {第一步指令}
2. {第二步指令}
3. {第三步指令}

## 示例
- 输入: {示例1}
- 输出: {期望输出1}

- 输入: {示例2}
- 输出: {期望输出2}
```
