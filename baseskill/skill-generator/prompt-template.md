# Skill Generator - Prompt 模板

## 会话内容
{session_content}

## 生成要求

### 1. 分析会话
请仔细阅读上面的会话内容，识别：
- 用户提出的核心问题或需求
- AI 提供的解决方案和回答
- 关键技术点或知识点
- 隐含的工作流程或方法论

### 2. 提取信息
从会话中提取以下信息：

**核心主题**: {topic}

**关键知识点**:
- {key_point_1}
- {key_point_2}
- {key_point_3}

**技能名称建议**: {suggested_skill_name}

**一句话描述**: {suggested_description}

### 3. 生成 Skill 草稿

请按照以下格式生成 Skill 草稿：

```yaml
---
name: {技能名称}
description: {一句话描述，20-50字}
---

# {技能名称}

## 使用时机
当用户需要{具体场景描述}时使用此技能。

## 指令
1. {第一条指令，具体可执行}
2. {第二条指令}
3. {第三条指令（可选）}

## 约束条件（可选）
- {约束1}
- {约束2}

## 示例
**示例 1**:
- 输入: {用户可能输入的内容}
- 输出: {期望的输出结果}

**示例 2**:
- 输入: {另一种输入情况}
- 输出: {对应输出}
```

### 4. 输出要求

- **name**: 简短格式，使用 kebab-case，如 `python-file-reader`
- **description**: 一句话说明功能，20-50字
- **instructions**: 3-5条清晰指令，每条10-30字
- **examples**: 提供2-3个输入输出示例

## 生成结果

请直接输出生成的 Skill 草稿内容，使用 YAML 格式。
